# 📖 Overview

IntelliSOC is an AI-powered Agentic Security Operations Center designed to simulate modern cybersecurity investigation workflows.

The platform combines a React frontend with a FastAPI backend to provide security analysts with an interactive environment for investigating security events, analyzing attack patterns, and understanding incident response workflows using realistic cybersecurity log simulations.

The project demonstrates how AI-assisted systems can improve threat investigation, reduce analysis time, and support Security Operations Center (SOC) teams.

---

# 🎯 Problem Statement

Modern Security Operations Centers generate thousands of security alerts every day.

Manual investigation is often:

- Time-consuming
- Repetitive
- Error-prone
- Difficult to scale

IntelliSOC demonstrates how AI-powered workflows can assist analysts by organizing security events, correlating attack indicators, and providing structured investigation capabilities.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/Tanyyy-27/intelliSOC.git
```

```
cd intelliSOC
```

---

# ⚙ Backend Setup

```
cd backend
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run server

```bash
uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

Backend runs at

```
http://127.0.0.1:8000
```

---

# 💻 Frontend Setup

```
cd frontend
```

Install packages

```bash
npm install
```

Run development server

```bash
npm run dev
```

Frontend runs at

```
http://127.0.0.1:5173
```
