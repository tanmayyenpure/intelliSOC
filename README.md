# 📖 Overview

IntelliSOC is an AI-Based Agentic Security Operations Center designed to simulate modern cybersecurity investigation workflows through intelligent automation and interactive threat analysis.

The platform combines a modern React frontend with a FastAPI backend to provide security analysts with an intuitive environment for investigating security events, analyzing attack patterns, and understanding incident response workflows using realistic cybersecurity datasets.

By integrating Artificial Intelligence with Security Operations Center (SOC) methodologies, IntelliSOC demonstrates how intelligent systems can reduce investigation time, improve analyst efficiency, and support faster security decision-making.

---

# 🎯 Problem Statement

Modern Security Operations Centers generate thousands of security alerts every day, making manual investigation increasingly difficult and resource intensive.

Traditional investigation workflows are often:

- Time-consuming
- Repetitive
- Error-prone
- Difficult to scale
- Dependent on manual correlation
- Challenging for new analysts

IntelliSOC addresses these challenges through AI-assisted investigation workflows, structured threat analysis, and an interactive analyst experience that simplifies cybersecurity operations.

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/tanmayyenpure/intelliSOC.git

cd intelliSOC
```

---

# ⚙️ Backend Setup

Navigate to the backend directory

```bash
cd backend
```

Install project dependencies

```bash
pip install -r requirements.txt
```

Start the FastAPI development server

```bash
uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

Backend will be available at

```
http://127.0.0.1:8000
```

---

# 💻 Frontend Setup

Navigate to the frontend directory

```bash
cd frontend
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Frontend will be available at

```
http://127.0.0.1:5173
```

# 🔍 Security Investigation Scenarios

The project includes realistic cybersecurity datasets that simulate modern Security Operations Center (SOC) investigations.

### 🔐 Credential Compromise

Investigate suspicious authentication activity, unusual login patterns, and potential account takeover attempts.

### 🎣 Phishing Investigation

Analyze malicious email campaigns, identify phishing indicators, and understand user interaction patterns.

### 🦠 Malware Beacon Detection

Detect periodic outbound communications, analyze beaconing behavior, and investigate compromised endpoints.

### 👤 Insider Threat Analysis

Investigate unauthorized internal access, privilege misuse, and suspicious user activities within an organization.
